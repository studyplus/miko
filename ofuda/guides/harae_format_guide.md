# harae.md フォーマットガイド

harae.md の構造とステータス管理のルールを定義する。

## 指摘のステータス

| ステータス | 意味 |
|---|---|
| **open** | 未対処 |
| **resolved** | ルール修正で対処済み |
| **dismissed** | 検討して問題なしと判断（理由付き） |
| **obsolete** | 関連ルールの廃止・改訂により指摘自体が無効化 |

**視覚的な区別:**
- **open**: ステータスに ❗ を付ける（例: `❗ open`）
- **open 以外**（resolved / dismissed / obsolete）: 指摘一覧テーブルの全セルを打ち消し線（`~~テキスト~~`）にする
